# Dashboard Northwind

## Descripción

Este proyecto consiste en el desarrollo de un dashboard interactivo utilizando Dash y Plotly para visualizar información de la base de datos Northwind.

## Tecnologías utilizadas

- Python
- Dash
- Plotly
- Pandas
- SQL Server
- PyODBC

## Funcionalidades

- Métricas generales
- Clientes por ciudad
- Productos por categoría
- Productos más caros
- Productos sin stock
- Filtro por categoría
- Filtro por precio mínimo

## Cómo ejecutar

Instalar dependencias:

```bash
pip install dash plotly pandas pyodbc
```

Ejecutar:

```bash
python app.py
```

Luego abrir:

```
http://127.0.0.1:8050/
```
